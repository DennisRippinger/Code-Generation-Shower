Example: jOOQ's Code Generator

Given: A database with a schema containing a table JOB_OFFER.

public List<JobOffer> findAll(Long companyID) {
  return jooq.select()
    .from(JOB_OFFER)
    .where(JOB_OFFER.COMPANY_ID.eq(companyID))
    .fetchInto(JobOffer.class);
}
